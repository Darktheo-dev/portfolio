Projects

ESP32 Smart Security Controller

I built a security monitoring system using an ESP32 microcontroller that detects motion and triggers alerts in real time. The device reads an IR sensor, communicates with a FastAPI backend, and updates a live dashboard so users can see activity the moment it happens. The system was designed with low latency, reliability, and simplicity in mind so it can be deployed in small-scale home or lab environments.

  • real-time motion detection using an IR sensor connected to the ESP32.
  • Instant alerting through API calls to a FASTAPI backend.
  • Web dashboard (React + Firebase) that displays sensor state and event history.

What I learned

hrough this project, I learned how to work with microcontroller GPIO, design reliable sensor logic, and send data quickly from the ESP32 to a FastAPI backend. I also built a real-time dashboard and improved my ability to debug embedded devices, handle async updates, and connect hardware, backend, and frontend systems smoothly.
